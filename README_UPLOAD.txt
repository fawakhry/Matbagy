نسخة v112 Mobile Send ZIP:

- إلغاء Web Share API لأنه غير ثابت على الموبايل.
- زر الإرسال الآن يعمل بثبات:
  1) يجهز ملفات الطباعة النظيفة HD 300DPI.
  2) يجمعها في ZIP واحد.
  3) ينزل ZIP على الموبايل.
  4) يفتح واتساب برسالة جاهزة.
  5) العميل يرفق ZIP من التنزيلات كـ Document / مستند.

ملاحظة مهمة:
المتصفح لا يستطيع إرفاق ملف تلقائيًا داخل واتساب عبر رابط wa.me، لذلك خطوة إرفاق الملف يدويًا ضرورية إلا لو استخدمنا سيرفر/Apps Script لرفع الملفات وإرسال روابط.
