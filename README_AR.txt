مطبعجي بنها - برنامج موبايل تجربة V1
====================================

هذه نسخة PWA تعمل على أندرويد من المتصفح ويمكن إضافتها للشاشة الرئيسية كتطبيق.

التشغيل السريع:
1) ارفع المجلد على أي استضافة أو افتح index.html للتجربة المحلية.
2) افتح الرابط من موبايل أندرويد باستخدام Chrome.
3) من قائمة Chrome اختر: Add to Home screen / إضافة إلى الشاشة الرئيسية.
4) كود التجربة: MB-1234 أو TEST أو 01000000000

المميزات:
- تفعيل أول مرة فقط.
- حفظ التفعيل على جهاز العميل.
- خلفية ودعاية خفيفة باسم مطبعجي بنها.
- اختيار قوالب: 6×9 / 10×15 / 7×10.
- فاصل 1 مم بين الصور.
- إنشاء شيتات 29.7×45 سم بدقة 300 DPI.
- وضع أمان كامل بدون قص للملامح.
- تدوير يدوي وتدوير تلقائي حسب القالب.
- تحميل الشيتات أو مشاركتها من الموبايل.
- طلب صلاحية إشعارات للتجهيز مستقبلاً.

ربط التفعيل بجوجل شيت:
1) أنشئ Google Sheet باسم Customers.
2) الأعمدة:
   Code | Phone | Name | Status
3) افتح Apps Script والصق ملف google_apps_script_activation.gs.
4) Deploy كـ Web App.
5) انسخ الرابط وضعه في config.js داخل activationEndpoint.

ملاحظات مهمة:
- الإشعارات الحقيقية Push تحتاج Firebase Cloud Messaging أو سيرفر Push لاحقاً.
- الإرسال المباشر لواتساب بالملفات يعتمد على دعم Web Share في جهاز العميل.
- النسخة الحالية آمنة للملامح لأن الوضع الافتراضي هو Fit بدون قص.
- لاحقاً يمكن تحويلها إلى APK Native أو WebView Android.
